U tvornici igračaka proizvode se gumene patkice, loptice i rubikove kocke. One su modelirane klasama Toy, Duck, Ball i Cube, kao na slici:

Tvornica proizvedene igračke privremeno pohranjuje u kutije. Vaš zadatak je modelirati generičku kutiju ToyBox u koju je moguće smjestiti neku vrstu igračke (ali ne i nešto što nije igračka). Kutija sadrži atribut toys tj. listu svih igračaka u njoj i atribut maxWeight koji predstavlja maksimalnu dopuštenu težinu kutije (int, zadaje se u konstruktoru). Dodatno, ToyBox ima konstruktor s jednim argumentom (maxWeight, tipa integer, postavlja se u konstruktoru i nije ga moguće naknadno promijeniti) te metode put i get, koje omogućavaju ubacivanje i vađenje igračaka iz njih. Metoda put prima jednu igračaku, sprema ju u internu listu (nije bitno gdje) i ne vraća ništa, dok metoda get nema argumenata, a vraća jednu igračku iz interne liste (nije bitno s koje pozicije). U slučaju da nije moguće obaviti naredbe put ili get (npr. ubacivanjem nove igračke kutija bi postala preteška ili u slučaju naredbe get, kutija je prazna), obje trebaju baciti neprovjeravanu iznimku tipa ToyBoxOperationException, s porukom "Box is full" ili "Box is empty".

public void put (T t);
public T get ();

Nadalje, postoji i specifična vrsta kutije, a to je CubeBox, u koju je moguće ubaciti samo Rubikove kocke. Ona (osim maxWeight koji je naslijedila) sadrži i varijablu maxVolume tipa int te je potrebno napisati odgovarajuću implementaciju metode put (a možda i get) koja i to uzima u obzir. CubeBox ima i odgovarajući konstruktor koji prima maxWeight i maxVolume (isto tako ih nije moguće naknadno promijeniti).

Konačno, klasa ToyBoxOperationException je klasa neprovjeravanih iznimki i mora sadržavati sve što je potrebno kako bi se mogle prikazati tražene poruke.

Primjer korištenja:

		Duck d1 = new Duck(10, "yellow"); //weight, color
		Duck d2 = new Duck(10, "red");    //weight, color
		Duck d3 = new Duck(990, "black"); //weight, color
		
		ToyBox<Duck> duckBox = new ToyBox<Duck>(1000);
		duckBox.put(d1); // ok
		duckBox.put(d2); // ok
		//duckBox.put(d3); // exc (ToyBox.put)
		duckBox.get(); // ok
		duckBox.put(d3); // ok
		duckBox.get(); // ok
		duckBox.get(); // ok
		//duckBox.get(); // exc (ToyBox.get)
		
		Cube c1 = new Cube(10, 10);  //weight, width (volume = 1000)
		Cube c2 = new Cube(10, 15);  //weight, width (volume = 3375)
		Cube c3 = new Cube(999, 10); //weight, width (volume = 1000)
		
		CubeBox cubeBox = new CubeBox(1000, 4000);
		
		cubeBox.put(c1); // ok
		//cubeBox.put(c2); // exc volume (CubeBox.put)
		//cubeBox.put(c3); // exc weight (ToyBox.put)
		cubeBox.get();
		cubeBox.put(c3); // ok
		cubeBox.get();
		cubeBox.put(c2); // ok
