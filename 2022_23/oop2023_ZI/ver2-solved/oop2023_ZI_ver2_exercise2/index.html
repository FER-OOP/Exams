U zadatku je potrebno nadopuniti programski kod klasa Triple i SimpleTripleTreeSet.

Klasa Triple predstavlja kolekciju od tri Stringa, na način da su u klasi definirani prvi, drugi i treći element, ali tako da su instance klase Triple međusobno jednake neovisno o poretku elemenata. Dakle, Triple s elemetima ("T","H","M") bi bio jednak ("H","M","T"). Također, Triple može sadržavati null elemente i duplikate.

Uz metode označene u UML dijagramu primjetite da klasa Triple treba implementirati sučelje Comparable. Za usporedbu koji je od dvije instance klase Triple veći, potrebno je njihove elemente sortirati prema prirodnom poretku, a onda usporediti najmanji element prve instance s najmanjim druge instance. U slučaju jednakosti, uspoređuju se srednji elementi dviju instanci, a ponovo u slučaju jednakosti, uspoređuju se najveći elementi. Pri tome ne smijete mijenjati privatne varijable first, second i third.

SimpleTripleTreeSet predstavlja skup elemenata tipa Triple. Za uspješnu implementaciju, SimpleTripleTreeSet treba čuvati sortiran skup elemenata prema njihovom prirodnom poretku. Skup po definiciji ne smije pohranjivati duplikate. Vaš je zadatak da nadopunite klasu prema UML dijagramu bez korištenja postojećeg sučelja Set i postojećih implementacija sučelja Set (HashSet, TreeSet, ...). Umjesto toga, trebate koristiti implementaciju sučelja Map, koju ćete definirati kao privatnu varijablu internalMap. Klasa SimplePairHashSet smije imati samo tu jednu varijablu. Tražene metode klase SimpleTripleTreeSet trebaju imati identičnu funkcionalnost kao i istoimene metode klase TreeSet (https://docs.oracle.com/en/java/javase/17/docs/api/java.base/java/util/TreeSet.html).

Napomena: Sve klase i sučelja imaju vidljivost postavljenu na package private i nalaze se u paketu hr.fer.oop.triple. Pri implementaciji metoda u njima smijete koristiti dodatne pomoćne strukture podataka jer brzina izvođenja programskog koda nije bitna tj. programski kod ne mora biti optimiziran. Pozivanje bilo koje metode ne smije "uništiti" kolekciju (tj. pobrisati sve elemente kolekcije ili slično).

Primjer korištenja:

Triple triple1 = new Triple("Peter", "Peter", "Nancy");
Triple triple2 = new Triple("Marco", "Jacob", "Peter");
Triple triple3 = new Triple("Peter", "Jacob", "Marco");

System.out.println("triple1 compared larger than triple2: " + (triple1.compareTo(triple2) > 0));
System.out.println("triple2 compared equal to triple3: " + (triple2.compareTo(triple3) == 0));

SimpleTripleTreeSet simpleTripleTreeSet = new SimpleTripleTreeSet();
System.out.println("add triple1: " + simpleTripleTreeSet.add(triple1));
System.out.println("add triple2: " + simpleTripleTreeSet.add(triple2));
System.out.println("add triple3: " + simpleTripleTreeSet.add(triple3));

simpleTripleTreeSet.forEach(triple -> System.out.printf("(%s, %s, %s);", triple.getFirst(), triple.getSecond(), triple.getThird()));
System.out.println();

System.out.println("simpleTripleTreeSet size: " + simpleTripleTreeSet.size());
System.out.println("simpleTripleTreeSet contains triple2: " + simpleTripleTreeSet.contains(triple2));
System.out.println("simpleTripleTreeSet contains triple3: " + simpleTripleTreeSet.contains(triple3));
System.out.println("simpleTripleTreeSet remove triple3: " + simpleTripleTreeSet.remove(triple3));
System.out.println("simpleTripleTreeSet remove triple3: " + simpleTripleTreeSet.remove(triple3));
System.out.println("simpleTripleTreeSet size: " + simpleTripleTreeSet.size());
