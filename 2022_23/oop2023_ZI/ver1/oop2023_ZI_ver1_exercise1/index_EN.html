1st TASK

You need to implement a class MyFileVisitor which will count how many files with given extensions are there in a given diretory and all of its subdirectories. The class allso needs to find the largest such file and the total number of directories including the starting directory.

Class MyFileVisitor needs to either implement the interface FileVisitor or to extend the class SimpleFileVisitor.

Class MyFileVisitor should be placed in the package hr.fer.oop.ZI2023. The class needs to have public methods and a constructor with the following signatures:

  • MyFileVisitor(String[extensions) - a constructor that takes an array of extensions
  • int getNumFiles() - a method that return the number of files with the given extensions
  • int getNumFolders() - a method that return a total number of directories
  • long getLargestFileSize() - a method that returns the size of the largest file in bytes, with the given extensions

Example:

A following directory tree is given (file sizes are given in parentheses):

data
--d2
----d21
      008_S8_L001_R2_001.fastq (7759)
      009_S9_L001_R1_001.fastq (35107)
      012_S12_L001_R1_001.fastq (29196)
----d22
      004_S4_L001_R1_001.fastq.gz (2713)
      004_S4_L001_R2_001.fastq.gz (2817)
  concentration_table.txt (13654)
  primers_1.fasta (550)
  primers_2.fasta (550)
  thresholds.txt (5578)

A following java code is given (You have it the class MainClass)

	public static void main(String[] args) {
		String[] extensions = {".txt", ".gz"};
		String dirName = "resources/data";	
		Path path = Paths.get(dirName);
		
		MyFileVisitor visitor = new MyFileVisitor(extensions);
		
		try {
			Files.walkFileTree(path,  visitor);
			
			System.out.printf("Number of files: %d%n", visitor.getNumFiles());
			System.out.printf("Number of folders: %d%n", visitor.getNumFolders());
			System.out.printf("Largest file size: %d%n", visitor.getLargestFileSize());
		}
		catch (IOException e) {
			e.printStackTrace();
		}
	}

Expected output:

Number of files: 4
Number of folders: 3
Largest file size: 13654

The largest file should be concentration_table.txt found within the folder data. The are other larger files within the directory tree, but they have wrong extensions.
