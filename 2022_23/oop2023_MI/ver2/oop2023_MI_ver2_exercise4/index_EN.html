Task Description

The task is to create an object model for vehicle competition. Below one can find a UML class diagram which is used in this task and it is related to vehicles (Vehicle), cars (Car) and racing cars (RacingCar). Unless stated otherwise, all the types are required to place in the package hr.fer.oop.

Interface Duel specifies methods for the duel between two entities, the interface is parametrized with two types, e.g. A and B. Visibility of this element secures the public access. It has:

  • Method double rateA(A) which calculates and returns the rating for the A-th entitity. The access of this type is public.
  • Method double rateB(B) which calculates and returns the rating for the B-th entitity. This element is visible from all the classes.
  • Method String battle(A, B) which compares the ratings of two entities and returns the textual representation of the entitity which has greater rating. Returns "DRAW" if the ratings are equal. For comparing the method Double.compare() must be used. This element has the maximum visibility.

The class Competition represents competition which is parametrized in a way so that A have to be either Vehicle or of type derived from Vehicle and B have to be either RacingCar or of type derived from RacingCar. It implements Duel. This element is visible from all the classes. It contains:

  • Method double rateA(A) which returns the number of the name characters. Visibility of this element secures the public access.
  • Method double rateB(B) which returns the average value of the name length, nitro value and power value. Visibility of this element secures the public access.
  • Method String battle(A, B) which offers the implementation according to the specification from the interface. This element has the maximum visibility.

The class RacingCarCompetition models a racing car (i.e., RacingCar) competition (note: the class cannot be parametrized). This element is visible from all the classes. It extends the type Competition. It is composed of:

  • Method double rateA(RacingCar) which returns the average value of the name length, nitro value and power value. This element is visible from all the classes.
