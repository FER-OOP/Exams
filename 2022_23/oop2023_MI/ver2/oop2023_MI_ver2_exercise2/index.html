Opis zadatka

U zadatku je potrebno implementirati objektni model za video industriju čiji se UML dijagram nalazi ispod. Kod za Art, Person i Actor je spreman za uporabu. Ako nije drugačije specificirano, sve tipove je potrebno smjestiti u paket hr.fer.oop.

Sučelje Person reprezentira osobu. U sebi sadržava:

  • Metodu String getName() koja dohvaća ime. Pristup ovog tipa je javan.
  • Metodu int getBirthYear() koja dohvaća godinu rođenja. Element je vidljiv iz svih klasa.

Klasa Actor modelira entitet glumca.

Klasa Director specificira entitet režisera. Pristup ovog tipa je javan. U sebi sadržava:

  • Varijablu name koja reprezentira ime i tipa je String.
  • Varijablu birthYear koja specificira godinu rođenju i tipa je int.
  • Varijablu retired koja reprezentira oznaku je li režiser u mirovini i tipa je boolean.
  • Konstruktor Director(String, int, boolean) koji postavlja ime, godinu rođenja i status je li umirovljen.
  • Metodu String toString() koja vraća String oblika "Director '[A]' ([B], retired: [C])", pri čemu [A], [B] i [C] slijedno označavaju ime, godinu rođenja i status umirovljenja.
  • Metodu String getName() koja je getter.
  • Metodu int getBirthYear() koja je getter.
  • Metodu boolean isRetired() koja je getter.

Enumeracija Rating reprezentira ocjene. Element ima maksimalnu vidljivost.

Sučelje Art specificira umjetnost. Ima:

  • Metodu Person author() koja vraća autora. Element ima maksimalnu vidljivost.

Klasa Movie modelira film. Element je vidljiv iz svih klasa. Ima:

  • Varijablu actors koja reprezentira polje glumaca i tipa je Person[]. Element je vidljiv klasama iz istog paketa te klasama koje naslijeđuju taj element.
  • Varijablu rating koja specificira ocjenu i tipa je Rating. Element je vidljiv klasama iz istog paketa te klasama koje naslijeđuju taj element.
  • Varijablu director koja specificira režiser i tipa je Director. Element je vidljiv klasama iz istog paketa te klasama koje naslijeđuju taj element.
  • Varijablu title koja specificira naslov i tipa je String.
  • Konstruktor Movie(String, Director, Person[], Rating) koji postavlja naslov, režisera, glumce i ocjenu.
  • Metodu Person author() koja vraća autora koji je zapravo režiser. Element ima maksimalnu vidljivost.
  • Metodu String toString() koja vraća String oblika "Movie '[A]' by [B] rated [C]", pri čemu [A], [B] i [C] označavaju naslov, autora i ocjenu.

Klasa Documentary predstavlja dokumentarni film. Vidljivost ovog elementa omogućuje javni pristup. Sadrži:

  • Varijablu forProfit koja oblikuje oznaku je li dokumentarni film komercijalan i tipa je boolean.
  • Konstruktor Documentary(String, Director, Rating, boolean) koji postavlja naslov, režisera, ocjenu i oznaku je li film komercijalan. Pretpostaviti da dokumentarac ima prazno polje glumaca veličine 0.
  • Metodu String toString() koja vraća String oblika "Documentary '[A]' by [B] rated [C] (for profit: [D])", pri čemu [A], [B], [C] i [D] slijedno označavaju naslov, tekstualnu reprezentaciju autora, ocjenu i oznaku je li film komercijalan.
  • Metodu boolean isForProfit() koja je getter.

Tablica vidljivosti

Primjer izvršavanja:


        Director director = new Director("Robert Zemeckis", 1952, false);
        System.out.println(director.getName());
        System.out.println(director.getBirthYear());
        System.out.println(director.toString());

        Person[] actors = new Person[]{new Actor("Tom Hanks", 1956, 150), new Actor("Robin Wright", 1966, 350), new Actor("Gary Sinise", 1955, 85)};
        System.out.println(actors[0].toString());

        Movie movie = new Movie("Forrest Gump", director, actors, Rating.PG13);
        System.out.println(movie.author());
        System.out.println(movie.toString());

        Documentary documentary = new Documentary("Oceans", new Director("Jacques Perrin", 1941, true), Rating.G, false);
        System.out.println(documentary.author());
        System.out.println(documentary.toString());

        System.out.println("director instanceof Director: " + (director instanceof Director));
        System.out.println("director instanceof Person: " + (director instanceof Person));
        System.out.println("actors[0] instanceof Actor: " + (actors[0] instanceof Actor));
        System.out.println("actors[0] instanceof Person: " + (actors[0] instanceof Person));

        System.out.println("movie instanceof Movie: " + (movie instanceof Movie));
        System.out.println("movie instanceof Art: " + (movie instanceof Art));

        System.out.println("documentary instanceof Documentary: " + (documentary instanceof Documentary));
        System.out.println("documentary instanceof Movie: " + (documentary instanceof Movie));
        System.out.println("documentary instanceof Art: " + (documentary instanceof Art));
