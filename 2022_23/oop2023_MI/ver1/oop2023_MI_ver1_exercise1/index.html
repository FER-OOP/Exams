Opis zadatka

U zadatku je potrebno imlementirati objektni model za zgrade. Ako nije drugačije specificirano, sve tipove je potrebno smjestiti u paket hr.fer.oop.

Sučelje EnergyEfficiency reprezentira funkcionalnost energetske efikasnosti. Vidljivost ovog elementa omogućuje javni pristup. U sebi sadržava:

  • Metodu boolean investigateEnergyEfficiency() koja vraća true ako je zgrada energetski učinkovita, inače false. Pristup ovog tipa je javan.

Razred Building modelira zgradu. Nije moguće instancirati primjerke ove klase. Implementira EnergyEfficiency. Vidljivost ovog elementa omogućuje javni pristup. Ima:

  • Varijablu address koja oblikuje adresu i tipa je String.
    • Potrebno je onemogućiti promjenu inicijalne vrijednosti i to na razini prevodioca.
    • Element je vidljiv klasama iz istog paketa te klasama koje naslijeđuju taj element.
  • Varijablu numberOfApartments koja reprezentira broj stanova i tipa je int.
    • Pristup je moguć iz klasa istog paketa te iz klasa koje naslijeđuju taj element.
    • Potrebno je onemogućiti promjenu inicijalne vrijednosti i to na razini prevodioca.
  • Varijablu energyConsumption koja je potrošnja energija i tipa je int.
    • Potrebno je onemogućiti promjenu inicijalne vrijednosti i to na razini prevodioca.
    • Pristup je moguć iz klasa istog paketa te iz klasa koje naslijeđuju taj element.
  • Konstruktor Building(String, int, int) koji postavlja adresu, broj stanova i potrošnju energije. Element je vidljiv iz svih klasa.
  • Metodu String getAddress() koja je getter. Element ima maksimalnu vidljivost.
  • Metodu int getNumberOfApartments() koja je getter. Element ima maksimalnu vidljivost.
  • Metodu int getEnergyConsumption() koja je getter. Element ima maksimalnu vidljivost.
  • Metodu String calculateEnergyClass() koja vraća energetski razred zgrade na temelju potrošnje energije.
    • Metoda nema svoju implementaciju.
    • Element ima maksimalnu vidljivost.

Klasa ResidentialBuilding je stambena zgrada. Element ima maksimalnu vidljivost. Dijete je od Building. Sadrži:

  • Varijablu numberOfParkingSpots koja reprezentira broj parkirnih mjesta i tipa je int. Vidljivost ovog elementa omogućuje javni pristup.
  • Konstruktor ResidentialBuilding(String, int, int, int) koji postavlja adresu, broj stanova, potrošnju energije i broj parkirnih mjesta. Element je vidljiv iz svih klasa.
  • Metodu int getNumberOfParkingSpots() koja je getter. Vidljivost ovog elementa omogućuje javni pristup.
  • Metodu boolean hasFreeParkingSpots() koja vraća true ako je broj parkirnih mjesta veći od broja stanova, inače vraća false. Vidljivost ovog elementa omogućuje javni pristup.
  • Metodu String calculateEnergyClass() koja računa energetski razred na sljedeći način: ako je potrošnja energije manja od 50, razred je A, ako je manja od 90, razred je B, inače je C. Element je vidljiv iz svih klasa.
  • Metodu boolean investigateEnergyEfficiency() koja vraća true ako je njezin energetski razred A, inače false. Pristup ovog tipa je javan.

Primjer izvršavanja:



       
        ResidentialBuilding rBuilding = new ResidentialBuilding("Address 1", 20, 500, 30);

        System.out.println("RESIDENTIAL BUILDING");
        System.out.println("Address: " + rBuilding.getAddress());
        System.out.println("Number of apartments: " + rBuilding.getNumberOfApartments());
        System.out.println("Number of parking spots: " + rBuilding.getNumberOfParkingSpots());
        System.out.println("Energy consumption: " + rBuilding.getEnergyConsumption());
        System.out.println("Energy class: " + rBuilding.calculateEnergyClass());
        System.out.println("Is energy efficient: " + rBuilding.investigateEnergyEfficiency());
        System.out.println("Has free parking spots: " + rBuilding.hasFreeParkingSpots());
