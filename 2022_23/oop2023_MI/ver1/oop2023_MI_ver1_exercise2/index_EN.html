Task Description

The task is to implement an object model for vehicles for which the UML class diagram is available below. Note: the code for Car, MotorVehicle and Trailer is available for use. Unless stated otherwise, all the types are required to place in the package hr.fer.oop.

Enumeration FuelType specifies fuel types. Visibility of this element secures the public access.

Interface MotorVehicle specifies vehicle functionality. It is composed of:

  • Method int getHorsePower() which returns the horse power. This element is visible from all the classes.
  • Method FuelType getFuelType() which returns fuel type. The access of this type is public.
  • Method String getModel() which returns the vehicle model. Visibility of this element secures the public access.

The class Car serves as a car.

The class Limousine is a limousine. This element is visible from all the classes. It has:

  • Variable passengerCapacity which specifies a passenger capacity and is of type int.
  • Constructor Limousine(String, String, int, FuelType, int, int) which sets the manufacturer, model, year of manufacture, fuel type, the number of horse power and passenger capacity.
  • Method String toString() which returns String with the pattern "[A] [B] [C] with [D] hp and with the passenger capacity of [E]", where [A], [B], [C], [D] and [E] represents the manufacturer, model, year of manufacture, the number of horse power and the passenger capacity, respectively.

Interface Trailer models the trailer functionality. It is composed of:

  • Method void attachTo(MotorVehicle) which attaches the trailer to the vehicle. This element has the maximum visibility.

The class Camper represents a camper entity. Visibility of this element secures the public access. It contains:

  • Variable attachedTo which specifies the vehicle that has attached this camper and is of type MotorVehicle. The element has the private visibility.
  • Constructor Camper() which initilizes the object.
  • Method String toString() which returns "Camper not attached to any motor vehicle" if there is no vehicle that attached this trailer, otherwise returns "Camper attached to '[A]'", where [A] denotes the vehicle model.

Sample code:


        MotorVehicle car = new Car("Opel", "Astra", 2007, FuelType.DIESEL, 113);
        System.out.println(car.getHorsePower());
        System.out.println(car.getFuelType());
        System.out.println(car.toString());

        Trailer camper = new Camper();
        System.out.println(camper.toString());
        camper.attachTo(car);
        System.out.println(camper.toString());

        MotorVehicle limousine = new Limousine("Lincoln", "Towncar", 2003, FuelType.GASOLINE, 205, 8);
        System.out.println(limousine.getHorsePower());
        System.out.println(limousine.getFuelType());
        System.out.println(limousine.toString());

        System.out.println("car instanceof Car: " + (car instanceof Car));
        System.out.println("car instanceof MotorVehicle: " + (car instanceof MotorVehicle));
        System.out.println("camper instanceof Camper: " + (camper instanceof Camper));
        System.out.println("camper instanceof Trailer: " + (camper instanceof Trailer));
        System.out.println("limousine instanceof Limousine: " + (limousine instanceof Limousine));
        System.out.println("limousine instanceof Car: " + (limousine instanceof Car));
        System.out.println("limousine instanceof MotorVehicle: " + (limousine instanceof MotorVehicle));
