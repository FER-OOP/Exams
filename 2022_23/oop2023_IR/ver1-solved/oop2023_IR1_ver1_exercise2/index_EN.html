Task Description

The task is to implement the reservation system for the hotel. Important: assume that the number 1 represents the first room. The methods must also work correct in the cases they are called with parameters that make no sense (e.g. negative numbers, non-existent rooms etc.). Unless stated otherwise, all the types are required to place in the package hr.fer.oop.

Interface ReservationSystem specifies functionalities related to the reservation system. Implementations of this interface must have a public constructor which has one parameter (int) that initializes the total number of rooms. It contains:

  • Method boolean checkIfRoomAvailable(int) which returns 'true' if the requested room number is available for booking, 'false' otherwise. The access of this type is public.
  • Method boolean bookRoom(int, int) which based on the room number and number of guests parameters, the system attempts to book a room. Reservation (success) is possible only if the room is available, in which case the system also tracks the number of people who will be in the room. Succesfull booking returnes 'true', 'false' othwerise. It is not possible to book a room without at least one person. This element is visible from all the classes.
  • Method boolean cancelReservation(int) which cancels the reservation based on the room number and returns the outcome: 'true' if succesful cancelation, 'false' otherwise. This element has the maximum visibility.
  • Method boolean modifyReservation(int, int) which modifies the room reservation based on the room number and the desired number of persons. Returns 'true' only if there was a change in the existing reservation, 'false' otherwise. This element is visible from all the classes.
  • Method int obtainAvailableRoomsCount() which returns the number of available rooms. This element is visible from all the classes.
  • Method String retrieveGuestList() which returns String which contains the information on the booked rooms. Rooms are divided with ';' where as each room is according to the format "Room [X]:[Y]", where [X] is the room number and [Y] are the number of people in the room. For example: "Room 1:5;Room 3:1" tells that the first room has 5 people whereas the third room has one person. The output must be ascending by the room number. Returns the empty String if there are no booked rooms. This element is visible from all the classes.

The class Solution represents the exercise solution that needs to be prepared (its direct superclass is the Object class). The element has the package-private visibility. It implements ReservationSystem.

Sample code:


        int totalRooms = 10;
        ReservationSystem reservationSystem = new Solution(totalRooms);

        System.out.println("Is room 1 available? " + reservationSystem.checkIfRoomAvailable(1));
        System.out.println("Available rooms count: " + reservationSystem.obtainAvailableRoomsCount());

        boolean booked = reservationSystem.bookRoom(2, 2);
        System.out.println("Booking in progress... result for room 2? " + booked);

        booked = reservationSystem.bookRoom(5, 1);
        System.out.println("Booking in progress... result for room 5? " + booked);

        booked = reservationSystem.bookRoom(8, 3);
        System.out.println("Booking in progress... result for room 8? " + booked);

        System.out.println("Is room 2 available? " + reservationSystem.checkIfRoomAvailable(2));
        System.out.println("Available rooms count: " + reservationSystem.obtainAvailableRoomsCount());

        boolean modified = reservationSystem.modifyReservation(5, 2);
        System.out.println("Modified reservation for room 5? " + modified);


        System.out.println("Is room 5 available? " + reservationSystem.checkIfRoomAvailable(5));

        boolean canceled = reservationSystem.cancelReservation(3);
        System.out.println("Canceled reservation for room 3? " + canceled);


        System.out.println("Is room 3 available? " + reservationSystem.checkIfRoomAvailable(3));
        System.out.println("Available rooms count: " + reservationSystem.obtainAvailableRoomsCount());

        booked = reservationSystem.bookRoom(7, 1);
        System.out.println("Booking in progress... result for room 7? " + booked);

        System.out.println("Is room 7 available? " + reservationSystem.checkIfRoomAvailable(7));
        System.out.println("Available rooms count: " + reservationSystem.obtainAvailableRoomsCount());

        System.out.println("Guest List:\n" + reservationSystem.retrieveGuestList());
