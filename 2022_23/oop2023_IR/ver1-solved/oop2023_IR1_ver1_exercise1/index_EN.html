Task Description

Create a Java program that analyses the CO2 emissions of the transport system with bycicles and cars. Unless stated otherwise, all the types are required to place in the package hr.fer.oop.

Interface Transport specifies operations on transport devices. Visibility of this element secures the public access. It is composed of:

  • Method String move() which makes the transport move end returns the textual representation of the move. The access of this type is public.
  • Method double calculateCO2(double) which calculate the amount of CO2 emissions (kg) produced by the transport over a certain distance (in kilometers). This element is visible from all the classes.

The class AbstractTransport specifies an abstract transport device. It implements Transport. The element has the package-private visibility. It is not possible to instantiate the objects of this class. It contains:

  • Variable name which represents the name of the transport and is of type String. This element is visible only from its own class.
  • Constructor AbstractTransport(String) which initializes the name. The access of this type is public.
  • Method String move() which returns "The [name] is moving.", where [name] is the name of the transport device. This element is visible from all the classes.
  • Method String toString() which returns "Transport [X]", where [X] is the name of the transportation device. This element has the maximum visibility.

The class Bicycle specifies a bicycle. Visibility of this element secures the public access. It is a child of AbstractTransport. It contains:

  • Variable numWheels which serves as the number of wheels on the bicycle and is of type int. The access is possible from the class of the same package and from the subclass.
  • Constructor Bicycle(String, int) which sets the name; sets the number of the wheels if greater than zero. This element has the maximum visibility.
  • Method double calculateCO2(double) which returns 0 because the bicycle does not produce CO2 emissions. This element has the maximum visibility.
  • Method String toString() which returns "Transport [X] is a bicycle with [Y] wheel(s)", where [X] is the name and [Y] are the number of wheels. This element is visible from all the classes.

The class Car represents a car. It extends the type AbstractTransport. The element has the package-private visibility. It contains:

  • Variable fuelEfficiency which models the fuel efficiency of the car [l/100 km] and is of type double. The access of this type is public.
  • Variable CO2_EMISSIONS which is CO2 (in kg) emissions per liter for an average car (value: 2.3) and is of type double.
    • The access of this type is public.
    • It is available in a static context of its own class.
    • It is important to prevent the modification of the initial value at the compiler level.
  • Constructor Car(String, double) which sets the name; sets the fuel efficiency if greater than zero, otherwise it is set to 10 [l/100 km]. The element is visible to the clases from the same package.
  • Method double calculateCO2(double) which calculates the amount of CO2 emissions (kg) produced by the car over a certain distance in km, based on its fuel efficiency and the amount of fuel consumed. Help using the example: a car with the fuel consupmtion of 10 l/100 km will consume 0.1 l/km. Next, if it drives 5 km then the consumption will be (0.1 l/km * 5 km) = 0.5 l. Consequently, CO2 emissions will be (0.5 l * 2.3 kg/l) = 1.15 kg. Visibility of this element secures the public access.
  • Method String toString() which returns "Transport [X] is a car that spends [Y] liters per 100 kilometers", where [X] is transport name and [Y] is the fuel efficiency with two decimal places. Visibility of this element secures the public access.

Sample code:


        Bicycle bike = new Bicycle("Bike A", 2);
        Car car = new Car("Car A", 7.4);
        
        AbstractTransport[] transportDevices = {bike, car}; 
        
        for(AbstractTransport device : transportDevices) {
        	System.out.println("-----");
        	System.out.println(device);
        	System.out.println(device.move());
        	System.out.printf("CO2 in kg for 10 kilometers: %.1f",device.calculateCO2(10));
        }

Expected output:

-----
Transport Bike A is a bicycle with 2 wheel(s)
The Bike A is moving.
CO2 in kg for 10 kilometers: 0.0-----
Transport Car A is a car that spends 7.40 liters per 100 kilometers
The Car A is moving.
CO2 in kg for 10 kilometers: 1.7
