U paketu hr.fer.oop.t3.items definirani su Item, Cloth, Beverage, Food, Milk, MilkType kao u primjerima s predavanja. Dodatno, definirana je i klasa FreshJuice kao još jedno kvarljivo piće.

Napomena: U testnim primjerima može biti i drugih klasa izvedenih iz navedene hijerarhije.

Klasa Box definirana je sljedećim kodom

public class Box<T> {
	protected List<T> content; //uočiti da je ovo protected
	public List<T> boxContent() {
		return Collections.unmodifiableList(content);
	}
	public Box(Collection<? extends T> elements) {
		content = new ArrayList<>(elements);
	}
}

Klasa Box je već napisana i njen kod se ne smije mijenjati!

Vaši zadaci su sljedeći:

  1. Napisati klasu MilkBox koja predstavja kutiju u kojoj se nalaze mlijeka. Kutija s mlijekom može nastati s kontruktorom čiji je jedini argument jedno mlijeko i ono čini početni sastav kutije. Nakon toga se dodatna mlijeka dodaju metodom add pri čemu se mora poštovati pravilo da se u kutiji mogu nalaziti samo mlijeka istog tipa. Ako se pokuša dodati mlijeko drugačijeg tipa nego što je prvo, mlijeko se neće dodati u kutiju i metoda će vratiti false. Ako je mlijeko uspješno dodano metoda treba vratiti true.

  2. Napisati klasu ItemsUtil koja će imati sljedeće statičke metode

    • totalVolume koja se može primijeniti na kutije čiji su elementi pića (dakle,kao argument možemo poslati Box<Beverage>, Box<Milk>, Box<FreshJuice> , ..., ali ne npr. Box<Food> ili Box<Perishable>). Metoda mora vratiti ukupan volumen pića u kutiji, odnosno 0 ako je sadržaj kutije prazan.
    • useBefore koja se može primijeniti na kutije čiji su elementi kvarljiva ruba (dakle,kao argument možemo poslati Box<Food>, Box<Milk>, Box<Perishable> , ..., ali ne npr. Box<Beverage>). Metoda mora vratiti najraniji istek roka trajanja među elementima kutije ili null ako je kutija prazna.
