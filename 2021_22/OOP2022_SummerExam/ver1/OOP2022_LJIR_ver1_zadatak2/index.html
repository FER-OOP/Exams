2. Zadatak (12,5 bodova)

Potrebno je napisati dvije klase: FootballPlayer i PlayerComparator. Sve klase trebaju biti u paketu hr.fer.oop.ljir.zad2 i imaju vidljivost postavljenu na public.

Klasa FootballPlayer

Klasa FootballPlayer ima 6 članskih varijabli:

  • private String name
  • private String surname
  • private int shootingSkill
  • private int dribblingSkill
  • private int passingSkill
  • private int overallSkill

Trebate napisati konstruktor koji ima 5 argumenata: name, surname, shootingSkill, dribblingSkill i passingSkill. Atribut overallSkill se računa kao zbroj ostala 3 skilla te se cijelobrojno dijeli s 3 ( (shootingSkill + dribblingSkill + passingSkill) / 3). Dodatno, potrebno je napisati i sve gettere i settere za svaki atribut.

Klasu FootballPlayer morate pripremiti za rad s kolekcijama tako da nadjačate metode equals i hashCode.

Metoda equals vraća da su dva igrača ista ako imaju isto ime i isto prezime.

Isto tako metoda hashCode će vratiti isti sažetak za dva igrača koja imaju isto ime i isto prezime.

Potrebno je također preurediti metodu toString kako bi odgovarala ispisu koji je prikazan u primjeru.

Klasa PlayerComparator

Potrebno je napisati klasu PlayerComparator koja implementira sučelje Comparator te uspoređuje dva igrača prvo po atributu overallSkill(manji ide prvo), zatim po prezimenu (abecedno) i na kraju po imenu (abecedno). Ne očekuje da će se komparator pozivati s vrijednostima null.

Primjer korištenja:

FootballPlayer player1 = new FootballPlayer("Luka","Modric", 90, 80, 100);
FootballPlayer player2 = new FootballPlayer("Ivan", "Modric", 90, 80, 100);
FootballPlayer player3 = new FootballPlayer("Stipe", "Modric", 60, 60, 60);
FootballPlayer player4 = new FootballPlayer("Luka", "Modric", 30, 30, 30);

System.out.println(player1);

System.out.println(player1.equals(player4));
System.out.println(player1.hashCode() == player4.hashCode());

List<FootballPlayer> playerList = new ArrayList<FootballPlayer>();
playerList.add(player1);
playerList.add(player2);
playerList.add(player3);
playerList.add(player4);

System.out.println("Before sorting: " + playerList);

PlayerComparator playerComparator = new PlayerComparator();
Collections.sort(playerList, playerComparator);

System.out.println("After sorting: " + playerList);

Rezultat:

true
true
Before sorting: [Luka Modric - 90, Ivan Modric - 90, Stipe Modric - 60, Luka Modric - 30]
After sorting: [Luka Modric - 30, Stipe Modric - 60, Ivan Modric - 90, Luka Modric - 90]
