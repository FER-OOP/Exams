2. zadatak (7 bodova)

Vaš zadatak je napisati programski kod na način kako je to specificirano UML dijagramom klasa na slici.

Legenda vidljivosti u dijagramu klasa:

Klasom Wallet je oblikovan digitalni novčanik koji može sadržavati 0 ili više žetona (tj. Token-a). Maksimalan broj žetona se nalazi u argumentu konstruktora count. Ako je count izvan raspona onda se pretpostavlaj da je 1. U metodi setToken parametar index je indeks u članskoj varijabli Token[]. Ako je indeks izvan raspona ova metoda ništa ne radi.

Apstraktna klasa Token predstavlja određenu kriptovalutu ili nerazmjenjivi žeton (eng. non-fungible token - NFT). Članskim varijablama amount i name je definiran iznos i ime Token-a. Apstraktnu metodu getTokenType() je potrebno nadjačati u izvedenim klasama na način da vraća string Cryptocurrency (ako se radi o kriptovaluti) ili string NFT (ukoliko se radi o NFT-u).

Tip kriptovalute je definiran enumeracijom CryptocurrencyType koja može imati dvije vrijednosti BTC i ETH. Vrijednost članske varijable exchangeAmount u klasi Cryptocurrency predstavlja tečaj određene kriptovalute u EUR-ima i postavlja se u konstruktoru klase Cryptocurrency generiranjem slučajnog broja u intervalu:

  • [40400, 40900] za tip kriptovalute BTC,
  • [2900, 3000] za tip kriptovalute ETH.

Ime kriptovalute je BTC ili ETH.

Dohvaćanje generiranog tečaja određene kriptovalute specificirano je sučeljem Exchangable na način da se implementitra metoda getExchangeRate() u odgovarajućim klasama.

Pozivom metode toString() u klasama Cryptocurrency i NFT se vraća tekst s ispisom iznosa i imena određene kriptovalute odnosno NFT-a. Primjer ispisa je ispod u primjeru korištenja.

Metoda printWallet() u klasi Wallet ispisuje sadržaj novčanika. Ako je na nemom mjestu žeton null taj ispis se preskače. Za žetone tipa Exchengable tekst ispiše se iznos kriptovalute i njene protuvrijednosti u EUR-ima u sljdećem formatu (koristi se poziv metode toString za ispis sadržaja tokena nakon čega se ispisuje protuvrijendost u EUR i tip tokena):

  • 0.0230 BTC equals 934.9040 EUR, token type: Cryptocurrency
  • 17.5000 ETH equals 51117.5000 EUR, token type: Cryptocurrency

Za ostale žetone metoda printWallet() ispisuje tekst s iznosom i imenom/tpom u sljdećem formatu (zadržaj i tip tokena):

  • 3 CryptoKitties, token type: NFT
  • 7 Doodles, token type: NFT

Primjer korištenja:

Wallet wallet = new Wallet(4);

wallet.setToken(new Cryptocurrency(0.023, CryptocurrencyType.BTC), 0);
wallet.setToken(new NFT(3, "CryptoKitties"), 1);
wallet.setToken(new Cryptocurrency(17.5, CryptocurrencyType.ETH), 2);
wallet.setToken(new NFT(7, "Doodles"), 3);

wallet.printWallet();

System.out.println("---");
System.out.println(new Cryptocurrency(0.023, CryptocurrencyType.BTC));
System.out.println(new NFT(3, "CryptoKitties"));
