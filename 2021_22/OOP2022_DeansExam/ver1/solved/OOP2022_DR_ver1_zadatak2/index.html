2. Zadatak (12,5 bodova)

Potrebno je napisati dvije klase: BasketballPlayer i BasketballPlayerComparator. Sve klase trebaju biti u paketu hr.fer.oop.zad2 i imaju vidljivost postavljenu na public.

Klasa BasketballPlayer

Klasa BasketballPlayer ima 6 članskih varijabli:

  • private String name
  • private String surname
  • private double pointsAvg
  • private double assistsAvg
  • private double reboundsAvg
  • private int position

Trebate napisati konstruktor koji prima 6 argumenata (sve gore navedene atribute).

Klasu BasketballPlayer morate pripremiti za rad s kolekcijama tako da nadjačate metode equals i hashCode.

Metoda equals vraća da su dva igrača ista ako imaju isto ime, isto prezime i igraju istu poziciju.

Isto tako metoda hashCode će vratiti isti sažetak za dva igrača koja imaju isto ime, isto prezime i igraju istu poziciju.

otrebno je također preurediti metodu toString kako bi odgovarala ispisu koji je prikazan u primjeru.

Klasa BasketballPlayerComparator

Potrebno je napisati klasu BasketballPlayerComparator koja implementira sučelje Comparator te uspoređuje dva igrača prvo po atributu position(manji ide prvo), zatim po atributima pointsAvg, assistsAvg i reboundsAvg (u svakoj usporedbi manji broj ide prije).

Primjer korištenja:

BasketballPlayer player1 = new BasketballPlayer("Ante", "Antic", 20.0, 10.0, 10.0, 3);
BasketballPlayer player2 = new BasketballPlayer("Ante", "Antic", 10.0, 20.0, 20.0, 3);
BasketballPlayer player3 = new BasketballPlayer("Borna", "Bornic", 10.0, 20.0, 20.0, 2);

System.out.println(player1.equals(player2));
System.out.println(player1.hashCode() == player2.hashCode());

List<BasketballPlayer> playerList = new ArrayList<BasketballPlayer>();
playerList.add(player1);
playerList.add(player2);
playerList.add(player3);

System.out.println("Before sorting: " + playerList);

BasketballPlayerComparator comparator = new BasketballPlayerComparator();
Collections.sort(playerList, comparator);

System.out.println("After sorting: " + playerList);

Rezultat:

true
true
Before sorting: [Ante Antic(3) [20.0, 10.0, 10.0], Ante Antic(3) [10.0, 20.0, 20.0], Borna Bornic(2) [10.0, 20.0, 20.0]]
After sorting: [Borna Bornic(2) [10.0, 20.0, 20.0], Ante Antic(3) [10.0, 20.0, 20.0], Ante Antic(3) [20.0, 10.0, 10.0]]
