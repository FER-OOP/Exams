3. zadatak (6 bodova)

Klase MyList<E>, MyListElement<E>, Media, Video, Audio, Podcast i Song zadane su kao na UML dijagramu.

Legenda vidljivosti u dijagramu klasa:

Napomena: Sve klase i sučelja imaju vidljivost postavljenu na public i nalaze se u paketu hr.fer.oop.zad3.

Napravite javno sučelje Printable koje ima jednu metodu print koja ne prima ništa, ali vraća String koji se može ispisati.

Napravite apstraktnu klasu PrintableList koja nasljeđuje klasu MyList i implementira sučelje Printable.

Napravite klasu MediaPrinter koja nasljeđuje klasu PrintableList i parametrizirana je tako da može sadržavati sve vrste medija. Metoda print vraća za svaki element jedan redak kako je u primjeru korištenja.

Napraviti klasu Printer. U njoj je potrebno napraviti 3 javne statičke metode:

  • metoda printVideoList koja vraća String, a prima listu MyList objekata iz klase Video. Metoda vraća po redak za svaki element liste. Format je u primjeru korištenja;
  • metoda printAudioList vraća String, a prima listu MyList objekata iz klase Audio. Metoda vraća po jedan redak po elementu liste. Format je u primjeru korištenja;

Primjer korištenja:

Video video1 = new Video("Dune", 9300, "H265", "MP3", "3840 x 2160");
Video video2 = new Video("Don't Look Up", 8280, "H264", "AAC", "1920 x 1080");

Podcast podcast2 = new Podcast("Coding 101", 1200, "AAC", "none");
Song song1 = new Song("Free software Song", 122, "MP3", "Richard Stallman");

Song song3 = new Song("I Hate the DMCA", 203, "AAC", "Joe Wecker");
