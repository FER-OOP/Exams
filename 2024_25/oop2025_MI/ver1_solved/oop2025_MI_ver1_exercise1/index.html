U zadatku je potrebno napisati klase kako je navedeno dijagramom klasa.

U svim klasama je potrebno i implementirati metode kako su prikazane na dijagramu klasa. Slobodno možete dodati i implementirati dodatne metode, atribute i konstruktore.

Sve klase i sučelja imaju vidljivost postavljenu na public i nalaze se u paketu hr.fer.oop.

Legenda vidljivosti u dijagramu klasa:

Sučelje Vehicle predstavlja vozilo i ima deklarirane dvije metode. Sučelje Green predstavlja vozilo koje za pokretanje koristi ekološku energiju. Ekološka energija je definirana u enumeraciji EnergySource i ima 3 vrijednosti.

Klasa AbstractVehicle predstavlja vozilo i ima implementaciju koja sprema zajedničke stvari za sva vozila. Postoje dvije podvrste:

  • Car - predstavlja automobil koji ima težinu i
  • Truck - predstavlja kamion koji vozi druge automobile. Kamion može voziti maksimalno 6 automobila, a sadrži težinu kamiona weight i nosivost loadCapacity (maksimalna težina svih automobila koji su na njega natovareni). U metodi add treba provjeravati te parametre i ako su prekoračeni kod dodavanja automobila onda ga ne smijemo staviti na kamion. Rezultat te metode je uspješnost stavljanja automobila na kamion. Metoda getCars služi za korištenje u potklasama i mora vraćati polje sadržanih automobila koje ako promijenimo ne utječe na promjenu sadržaja kamiona. Težina je težina vozila s teretom.

Osim toga imamo i klasu GreenCar koji predstavlja automobil koji koristi ekološku energiju. Isto tako imamo i GreenTruck koji predstavlja kamion koji koristi ekološku energiju. U toj klasi imamo i dodatnu metodu getWeightOfGreenCars() koja vraća težinu automobila koji koriste ekološku energiju.

Primjer korištenja:

// Create a Car instance
Car car1 = new Car("Sedan", "BMW", 1500);
Car car2 = new Car("SUV", "Audi", 1800);

// Print car details
System.out.println("Car 1: " + car1.getType() + ", " + car1.getCarBrand() + ", Weight: " + car1.getWeight());
System.out.println("Car 2: " + car2.getType() + ", " + car2.getCarBrand() + ", Weight: " + car2.getWeight());

// Create a Truck instance
Truck truck = new Truck("Truck", "Mercedes", 3000, 5000);

// Add cars to the truck
System.out.println("Adding car1 to truck: " + truck.add(car1));
System.out.println("Adding car2 to truck: " + truck.add(car2));

// Print the total weight of the truck
System.out.println("Total truck weight: " + truck.getWeight());

// Create a GreenCar instance
GreenCar greenCar = new GreenCar("Hatchback", "Tesla", 1200, EnergySource.ELECTRICITY);

// Print GreenCar details
System.out.println("GreenCar: " + greenCar.getType() + ", " + greenCar.getCarBrand() +
  ", Weight: " + greenCar.getWeight() + ", Energy Source: " + greenCar.getEnergySource());

// Create a GreenTruck instance
GreenTruck greenTruck = new GreenTruck("GreenTruck", "Volvo", 4000, 6000, EnergySource.BIOFUEL);

// Add cars to the green truck
System.out.println("Adding greenCar to greenTruck: " + greenTruck.add(greenCar));
System.out.println("Adding car1 to greenTruck: " + greenTruck.add(car1));
System.out.println("Adding car2 to greenTruck: " + greenTruck.add(car2));
System.out.println("Adding Porsche to greenTruck: " + greenTruck.add(
    new Car("Coupe", "Porsche", 1600)));

// Print the total weight of green cars in the green truck
System.out.println("Total weight of green cars in greenTruck: " + greenTruck.getWeightOfGreenCars());
