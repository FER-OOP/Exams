Opis zadatka

Imate zadatak pomoći u vođenju kvalifikacija za natjecanje STEM Games koje se provodi na FER-u. U kvalifikacijama svaki student dobiva bodove iz dvije kategorije: inženjerstvo ( engineering ) i matematika ( math ).

Potrebno je napisati dvije klase: Student i STEMGames. Sve klase trebaju biti u paketu hr.fer.oop.ZI2024.zad3 i imaju vidljivost postavljenu na public .

Klasa Student treba imate sljedeće privatne članske varijable i javne metode sa zadanim potpisima:

  • name : String - ime i prezime studenta
  • pointsEng : int - broj ostvarenih bodova iz inženjerstva
  • pointsMath : int - broj ostvarenih bodova iz matematike
  • Student(String name, int pointsEng, int pointsMath) - konstruktor koji prima ime studenta name, broj ostvarenih bodova iz kategorije inženjerstva pointsEng, i broj ostvarenih bodova iz kategorije matematika pointsMath
  • String getName() - metoda koja vraća prezime studenta
  • int getPointsEng() - metoda koja vraća broj ostvarenih bodova iz inženjerstva
  • int getPointsMath() - metoda koja vraća broj ostvarenih bodova iz matematike
  • int getTotalPoints() - metoda koja vraća ukupne bodove kao zbroj ostvarenih bodova iz inženjerstva i matematike

Ako želite, možete dodati i implementirati dodatne metode, atribute i konstruktore.

U klasi STEMGames potrebno je implementirati javne metode sa zadanim potpisima:

  • void addStudent(String name, int pointsEng, int pointsMath) - metoda koja dodaje novi zapis sa studentom u objekt STEMGames
  • List<Student> getStudents() - metoda koja vraća listu studenata
  • int getHighestScoreInEng() - metoda koja vraća najveći ostvareni broj bodova iz inženjerstva
  • int getHighestScoreInMath() - metoda koja vraća najveći ostvareni broj bodova iz matematike
  • int getHighestScoreInTotal() - metoda koja vraća najveći ostvareni ukupni broj bodova
  • Map<String, List<String>> getTopStudents() - metoda koja vraća mapu sa tri zapisa. Ključevi imaju vrijednosti eng, math i total, a vrijednosti su liste imena studenata koji su ostvarili najveći broj bodova prema kriteriju iz ključa.
