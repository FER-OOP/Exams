Task Description

You are tasked with helping to manage the qualifications for the STEM Games competition held at FER. In the qualifications, each student receives points from two categories: engineering and mathematics.

It is necessary to write two classes: Student and STEMGames. All classes should be in hr.fer.oop.ZI2024.zad3 package and have visibility set to public.

The Student class should have the following private member variables and public methods with these signatures:

  • name : String - student name
  • pointsEng : int - points in engineering
  • pointsMath : int - points in mathematics
  • Student(String name, int pointsEng, intpointsMath) - constructor that receives attributes student name name, number of achieved points from engineering category pointsEng, and the number of achieved points from the mathematics category pointsMath
  • String getName() - method that returns the name of the student
  • int getPointsEng() - a method that returns the number of achieved engineering points
  • int getPointsMath() - a method that returns the number of math points achieved
  • int getTotalPoints() - a method that returns total points as the sum of achieved points in engineering and mathematics

If you want, you can add and implement additional methods, attributes, and constructors.

In class STEMGames you need to implement following public methods:

  • void addStudent(String name, int pointsEng, int pointsMath) - a method adding a new student to STEMGames object
  • List<Student> getStudents() - a method returning a list of students
  • int getHighestScoreInEng() - a method returning the highest score in engineering
  • int getHighestScoreInMath() - a method returning the highest score in mathematics
  • int getHighestScoreInTotal() - a method returning the highest total score
  • Map<String, List<String>> getTopStudents() - a method returning a map with three entries. Keys have values eng, math and total, while values are lists of student names for students with a highscore according to a criteria specified by the key.
