Opis zadatka

Imate zadatak pomoći u vođenju kvalifikacija za natjecanje STEM Games koje se provodi na FER-u. U kvalifikacijama svaki student dobiva bodove iz dvije kategorije: inženjerstvo ( engineering ) i matematika ( math ).

Potrebno je napisati dvije klase: Student i STEMGames. Sve klase trebaju biti u paketu hr.fer.oop.ZI2024.zad3 i imaju vidljivost postavljenu na public .

Klasa Student treba imate sljedeće privatne varijable:

  • name : String . ime i prezime studenta
  • pointsEng : int - bodovi iz inženjerstva
  • pointsMath : int - bodovi iz matematike

U klasi Student potrebno je implementirati javne metode i konstruktor sa zadanim potpisima:

  • Student(String name, int pointsEng, int pointsMath) - konstruktor koji prima ime i prezime studenta name, broj ostvarenih bodova iz kategorije inženjerstva pointsEng, i broj ostvarenih bodova iz kategorije matematika pointsMath
  • String getName() - metoda koja vraća ime i prezime studenta (name))
  • int getPointsEng() - metoda koja vraća broj ostvarenih bodova iz inženjerstvaa
  • int getPointsMath() - metoda koja vraća broj ostvarenih bodova iz matematikee
  • int getTotalPoints() - metoda koja vraća ukupne bodove kao zbroj ostvarenih bodova iz inženjerstva i matematikee

Ako želite, možete dodati i implementirati dodatne metode, atribute i konstruktore.

U klasi STEMGames potrebno je implementirati javne metode sa zadanim potpisima:

  • void addStudent(String name, int pointsEng, int pointsMath) - metoda koja dodaje novi zapis sa studentom u objekt STEMGames
  • List<Student> getStudents() - metoda koja vraća listu studenta
  • int getTopEngStudentCount(int threshold) - metoda koja vraća broj studenata s bodovima iz inženjerstva jednakima ili većima od zadanog praga (threshold)
  • int getTopMathStudentCount(int threshold) - metoda koja vraća broj studenata s bodovima iz matematike jednakima ili većima od zadanog praga (threshold)
  • int getTopTotalStudentCount(int threshold) - metoda koja vraća broj studenata s ukupnim bodovima jednakima ili većima od zadanog praga (threshold)
  • Map<String, List<String>> getTopStudents() - metoda koja vraća mapu s tri zapisa. Ključevi imaju vrijednosti eng, math i total, a vrijednosti su liste imena studenata s odgovarajućim ostvarenim bodovima jednakima ili većima od zadanog praga. Prag za bodove iz inženjerstva i matematike je 95, a prag za ukupne bodove je 190.
