Your task is to implement a class ChessPlayer for storing information on chess players. Each chess players has the following attributes:

  • firstname : first name
  • lastname : last name
  • country : country of origin – countries are stored in enumeration Country which is already implemented
  • rating – ELO rating (integer)
  • birthyear – year of birth (integer)

All atributes should be private and for each of them you have to implement a getter. Be precise with getter names, e.g. a getter for the attribute firstname should be named getFirstname().

You also need to define a constructor which will create a new ChessPlayer object based on attribute values:

public ChessPlayer(String firstname, String lastname, Country country, int rating, int birthyear)

Override method toString() as follows:

public String toString() {
	return String.format("(%s) %s %s %d: %d", country, firstname, lastname, birthyear, rating);
}

Implement all additional method that are required for the class ChessPlayer to be used with java collections (ArrayList, HashSet, TreeSet). Take into account that two chess players are equal if their first name, last name and rating are equal. No two chess players can have same last name, first name and rating. They can have same first name and last name, but then rating has to be different. Define natural comparator which will sort chess players first by their last name (ascending), then by their first name (ascending) and last by their rating (descending).

Define public, static and final variables which will contain comparators for specific attributes:

  • BY_LASTNAME - comparator for the last name (ascending)
  • BY_FIRSTNAME – comparator for the first name (ascending)
  • BY_RATING – comparator for the rating (ascending - different from natural comparator)

IMPORTANT: Class ChessPlayerData is already implemented and can be used. It contains several methods for creating a list of ChessPlayer objects that can be used for testing. The class randomizes the order of the data, so the output order can sometimes be different than in the example. Grading tests will take that into account.

IMPORTANT: When implementing comparators take care that they work correctly for Croatian characters. A String comparator for Croatian language can be obtained by:

Collator.getInstance(Locale.forLanguageTag("hr"))

IMPORTANT: ClassChessPlayer should be public and placed in the package hr.fer.oop.ZI2024.zad2.

List<ChessPlayer> lplayers = ChessPlayerData.loadChessPlayersSmall();
