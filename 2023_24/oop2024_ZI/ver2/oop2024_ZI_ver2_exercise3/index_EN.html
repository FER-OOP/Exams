Task Description

You are tasked with helping to manage the qualifications for the STEM Games competition held at FER. In the qualifications, each student receives points from two categories: engineering and mathematics.

It is necessary to write two classes: Student and STEMGames. All classes should be in hr.fer.oop.ZI2024.zad3 package and have visibility set to public.

Class Student should have following private variables:

  • name : String - student name
  • pointsEng : int - points achieved in engineering
  • pointsMath : int - points achieved in mathematics

In the `Student' class, it is necessary to implement public methods and a constructor with these signatures:

  • Student(String name, int pointsEng, int pointsMath) - constructor that receives attributes student name name, number of achieved points from engineering category pointsEng, and the number of achieved points from the mathematics category pointsMath
  • String getName() - method that returns the name of the student
  • int getPointsEng() - a method that returns the number of engineering points achieved
  • int getPointsMath() - a method that returns the number of mathematics points achieved
  • int getTotalPoints() - a method that returns total points as the sum of achieved points in engineering and mathematics

If you want, you can add and implement additional methods, attributes, and constructors.

In class STEMGames you need to implement following public methods:

  • void addStudent(String name, int pointsEng, int pointsMath) - a method adding a new student to STEMGames object
  • List<Student> getStudents() - a method returning a list of added students
  • int getTopEngStudentCount(int threshold) - a method returning the number of students with score in Engineering above or equal to the given threshold
  • int getTopMathStudentCount(int threshold) - a method returning the number of students with score in Mathematics above or equal to the given threshold
  • int getTopTotalStudentCount(int threshold) - a method returning the number of students with total score above or equal to the given threshold
  • Map<String, List<String>> getTopStudents() - a method returning a map with three entries. Keys have values eng, math and total, while values are lists of student names for students with a specific score above or equal to given threshold. A threshold for engineering and mathematics is 95, while threshold for total points is 190.
