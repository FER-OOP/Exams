Potrebno je napisati klase koje opisuju bioinformatičke podatke. Podaci se sastoje od sekvenci DNA, RNA i proteina. DNA, RNA i proteini su dugački slijedovi manjih molekula. DNA i RNA se sastoje od nukleotida, a proteini od aminokiselina. Nukleotidi i aminokiseline se označavaju po jednim slovom, tako da se sekvence DNA, RNA i proteina mogu pohraniti kao String.

Osnovni razred Sequence opisuje općenitu sekvencu. Primjerke ovog razreda nije moguće stvoriti. Razred Sequence ima dva atributa tipa String, kojima nije moguće pristupati iz drugih razreda:

  • name : naziv sekvence
  • seq : sam slijed sekvence

Implementirati gettere za navedene atribute.

Razred Sequence definira konstruktor:

  • public Sequence(String name, String sequence)

kao i metodu:

  • public String toFASTA()

Koja vraća sekvencu u FASTA formatu. FASTA je tekstualni format koji svaku sekvencu opisuje u dva retka. Prvi redak počinje znakom '>', nakon kojeg dolazi ime sekvence. U drugom retku nalazi se slijed sekvence.

NAPOMENA: Za implementaciju klase Sequence možete dopuniti klasu iz 1. zadatka ili napraviti novu. Nije nužno rješiti prvi zadatak da bi mogli riješiti drugi.

Potrebno je definirati sučelje CodingSequence s metodom public String getProteinSequence(). To sučelje opisuje kodirajuće sekvence koje se mogu prevesti u proteine. Radi jednostavnosti metoda vraća String (a ne poseban objekt). Pretvaranje slijeda sekvence u proteinski slijed obavlja se pozivom statičke metode public static String getProteinForDNA(String DNASeq) klase SequenceOperations. Klasa SequenceOperations vam je već dana i ne treba je implementirati.

Potrebno je definirati i jednu dodatnu klasu GeneSequence koja implementira sučelje CodingSequence.

Klasa GeneSequence nasljeđuje Sequence i predstavlja DNA sekvencu jednog gena. Ima sljedeće dodatne privatne atribute:

  • location tipa int – predstavlja lokaciju gena na genomu
  • genomeName tipa String – predstavlja ime genoma

Za dodatne atribute napraviti gettere i settere.

Klasa GeneSequence također ima konstruktor: public GeneSequence(String name, String sequence, String genomename, int location).

Napomena: Sve klase imaju vidljivost postavljenu na public i nalaze se u paketu hr.fer.oop.genomeseq.

Primjer izvođenja:

	Sequence s1 = new GeneSequence("seq1", "ACGTGTACG");
	Sequence s2 = new GeneSequence("seq2", "ACGTGTAGTACGTGTGTGAACAAT");
		
	System.out.println(s1.toFASTA());
	System.out.println(s2.toFASTA());
		
	System.out.println(((GeneSequence)s1).getProteinSequence());
	System.out.println(((GeneSequence)s2).getProteinSequence());

Rezultat ispisa:

>seq1
ACGTGTACG

>seq2
ACGTGTAGTACGTGTGTGAACAAT

TCT
TCSTCVNN
