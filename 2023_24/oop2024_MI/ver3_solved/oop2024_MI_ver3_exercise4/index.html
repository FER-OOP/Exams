Potrebno je napisati klase koje opisuju bioinformatičke podatke. Podaci se sastoje od sekvenci DNA, RNA i proteina. DNA, RNA i proteini su dugački slijedovi manjih molekula. DNA i RNA se sastoje od nukleotida, a proteini od aminokiselina. Nukleotidi i aminokiseline se označavaju po jednim slovom, tako da se sekvence DNA, RNA i proteina mogu pohraniti kao String.

Osnovni razred Sequence opisuje općenitu sekvencu. Primjerke ovog razreda nije moguće stvoriti. Razred Sequence ima dva atributa tipa String, kojima nije moguće pristupati iz drugih razreda:

  • name : naziv sekvence
  • seq : sam slijed sekvence

Atribute nije moguće mijenjati nakon postavljanja kroz konstruktor:

  • public Sequence(String name, String sequence)

Za navedene atribute napisati i getter-e.

Napravite dva dodatna razreda koji nasljeđuju Sequence, DNASequence i RNASequence. Ti razredi nemaju dodatne atribute, definiraju metodu toString koja ispisuje odgovarajući tekst i naziv sekvence.

(Vidi primjer - drugačije nego u prvom zadatku, ispisuje se samo vrsta sekvence i naziv, bez slijeda).

Potrebno je implementirati parametriziranu klasu MySeqList<??????????> (parametrizaciju trebate sami dobro odrediti) u koju će se pohranjivati više sekvenci. Klasa MySeqList smije pohranjivati samo sekvence (klasa Sequence i nasljeđene klase). Da bi vam pomogli s tim, dana vam je već implementirana klasa MyList<T> sa sljedeći metodama:

  • public MyList() – konstruktor
  • public int size() – vraća veličinu liste
  • public void add(T element) – dodaje element na početak liste
  • public T get(int index) – vraća element na zadanoj poziciji (null ako takav element ne postoji)

U klasi MySeqList potrebno je implementirati dodatnu metodu:

  • public void addall(MySeqList<??????????> anotherlist) – dodaje sve elemente iz liste anotherlist, jedan po jedan, pomoću metode get.

Sami morate točno odrediti parametrizaciju klase MySeqList i metode addall.

Napomena: Sve klase imaju vidljivost postavljenu na public i nalaze se u paketu hr.fer.oop.genomeseq.

Primjer izvođenja:

DNASequence s1 = new DNASequence("seq1", "ACGTGCTGA");
DNASequence s2 = new DNASequence("seq2", "AGGGTGGA");

RNASequence s3 = new RNASequence("seq3", "GGTTGTGTCCTGCGCTA");
RNASequence s4 = new RNASequence("seq4", "GAGAT");

MySeqList<Sequence> l1 = new MySeqList<>();	
MySeqList<DNASequence> l2 = new MySeqList<>();
MySeqList<RNASequence> l3 = new MySeqList<>();

l2.add(s1);
l2.add(s2);
l3.add(s3);
l3.add(s4);

l1.addall(l2);
l1.addall(l3);

for (int i=0; i<l1.size(); i++) {
	Sequence s = l1.get(i);
	System.out.println(s);
}

Rezultat:

RNA sequence: seq3
RNA sequence: seq4
DNA sequence: seq1
DNA sequence: seq2
