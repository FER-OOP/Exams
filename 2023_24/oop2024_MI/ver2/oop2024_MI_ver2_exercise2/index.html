Opis zadatka

Napisati program koji modelira računala (prijenosno računalo i stolno računalo) te računa njihove ocjene. Ako nije drugačije specificirano, sve tipove je potrebno smjestiti u paket hr.fer.oop.

Razred Computer reprezentira računalo. Stvaranje objekata iz ove klase nije moguće. Vidljivost ovog elementa omogućuje javni pristup. Sadrži:

  • Varijablu model koja modelira model računala i tipa je String. Element je vidljiv iz svih klasa.
  • Varijablu mhz koja specificira brzinu procesoru u megahercima i tipa je double. Element ima privatnu vidljivost.
  • Konstruktor Computer(String, double) koji inicijalizira model i brzinu procesora u megahercima. Element je vidljiv iz svih klasa.
  • Metodu String getModel() koja je getter. Vidljivost ovog elementa omogućuje javni pristup.
  • Metodu void setModel(String) koja je setter. Pristup ovog tipa je javan.
  • Metodu double gigahertz() koja vraća brzinu u gigahercima. Element je vidljiv iz svih klasa.

Razred Laptop predstavlja prijenosno računalo. Potrebno je zabraniti nasljeđivanje ove klase. Element ima maksimalnu vidljivost. Dijete je od Computer. U sebi sadržava:

  • Varijablu batteryCapacity koja reprezentira kapacitet baterije i tipa je int. Element ima privatnu vidljivost.
  • Konstruktor Laptop(String, double, int) koji inicijalizira model, brzinu procesora u megahercima i kapacitet baterije. Element ima maksimalnu vidljivost.
  • Metodu int getBatteryCapacity() koja je getter. Element je vidljiv iz svih klasa.
  • Metodu void setBatteryCapacity(int) koja je setter. Element ima maksimalnu vidljivost.

Razred Desktop reprezentira stolno računalo. Potrebno je zabraniti nasljeđivanje ove klase. Podtip je od Computer. Element je vidljiv iz svih klasa. U sebi sadržava:

  • Varijablu location koja predstavlja lokaciju i tipa je String. Element je vidljiv klasama iz istog paketa te klasama koje naslijeđuju taj element.
  • Konstruktor Desktop(String, double, String) koji inicijalizira model, brzinu procesora u megahercima i lokaciju. Element je vidljiv iz svih klasa.
  • Metodu String getLocation() koja je getter. Element ima maksimalnu vidljivost.
  • Metodu void setLocation(String) koja je setter. Vidljivost ovog elementa omogućuje javni pristup.

Klasa Calculator je kalkulator računala. Element je vidljiv iz svih klasa. Ima:

  • Metodu double score(Computer) koja računa ocjenu računala na sljedeći način. Ako se radi o stolnom računalu, vraća prosjek između duljine znakova modela i brzine u megahercima. Ako se radi o laptopu, vraća prosjek između kapaciteta baterije i brzine u gigahercima. Važno: u bilo kojem drugom slučaju (npr. nepravilan unos) vraća 0.
    • Vidljivost ovog elementa omogućuje javni pristup.
    • Dostupna je u statičkom kontekstu svoje klase.

Primjer izvršavanja:


		Desktop d1 = new Desktop("BANJA-PC", 2500, "BANJA");
		Desktop d2 = new Desktop("MIRA-PC", 1800, "UNSKA 3");
		
		Laptop l1 = new Laptop("MEK", 3000, 10);
		Laptop l2 = new Laptop("HA-PE", 1000, 8);
		Laptop l3 = new Laptop("LEDOVO", 1400, 12);
		
		Computer[] computers = {d1, d2, l1, l2, l3};
		
		for(Computer computer : computers) {
			System.out.printf("Computer %s (%.2f Ghz) has a score of %.2f %n", computer.getModel(), computer.gigahertz(), Calculator.score(computer));
		}
