Task Description

Write a program that models computers (laptop and desktop) and calculates their scores. Unless stated otherwise, all the types are required to place in the package hr.fer.oop.

The class Computer specifies a computer. Creating objects from this class is not possible. Visibility of this element secures the public access. It contains:

  • Variable model which specifies a computer model and is of type String. This element is visible from all the classes.
  • Variable mhz which models a processor speed in megahertz and is of type double. The element has the private visibility.
  • Constructor Computer(String, double) which initializers the model and processor speed in megahertz. This element is visible from all the classes.
  • Method String getModel() which is getter. Visibility of this element secures the public access.
  • Method void setModel(String) which is setter. The access of this type is public.
  • Method double gigahertz() which returns the speed in gigahertz. This element is visible from all the classes.

The class Laptop models a laptop. It is required to prevent the inheritance of this class. This element has the maximum visibility. It is a child of Computer. It is composed of:

  • Variable batteryCapacity which models the battery capacity and is of type int. The element has the private visibility.
  • Constructor Laptop(String, double, int) which initializes the model, the processor speed in megahertz and the battery speed. This element has the maximum visibility.
  • Method int getBatteryCapacity() which is getter. This element is visible from all the classes.
  • Method void setBatteryCapacity(int) which is setter. This element has the maximum visibility.

The class Desktop models a desktop computer. It is required to prevent the inheritance of this class. It is a subtype of Computer. This element is visible from all the classes. It is composed of:

  • Variable location which models a location and is of type String. The element is visible to clasess from the same paket as well as from the subclass.
  • Constructor Desktop(String, double, String) which initializes the model, speed and location. This element is visible from all the classes.
  • Method String getLocation() which is getter. This element has the maximum visibility.
  • Method void setLocation(String) which is setter. Visibility of this element secures the public access.

The class Calculator is a computer calculator. This element is visible from all the classes. It has:

  • Method double score(Computer) which calculates the score as follows. If desktop computer, returns the average of speed in megahertz and characters length of model. In case of laptop, return the average of battery capacity and processor speed in gigahertz. Important: in any other case (e.g. invalid input) returns 0.
    • Visibility of this element secures the public access.
    • It is available in a static context of its own class.

Sample code:


		Desktop d1 = new Desktop("BANJA-PC", 2500, "BANJA");
		Desktop d2 = new Desktop("MIRA-PC", 1800, "UNSKA 3");
		
		Laptop l1 = new Laptop("MEK", 3000, 10);
		Laptop l2 = new Laptop("HA-PE", 1000, 8);
		Laptop l3 = new Laptop("LEDOVO", 1400, 12);
		
		Computer[] computers = {d1, d2, l1, l2, l3};
		
		for(Computer computer : computers) {
			System.out.printf("Computer %s (%.2f Ghz) has a score of %.2f %n", computer.getModel(), computer.gigahertz(), Calculator.score(computer));
		}

Expected output:

Computer MEK (3.00 Ghz) has a score of 6.50 
Computer HA-PE (1.00 Ghz) has a score of 4.50
