Task Description

Write a program that models animals (dogs and cats) and calculates their scores. Unless stated otherwise, all the types are required to place in the package hr.fer.oop.

The class Animal specifies an animal. Creating objects from this class is not possible. Visibility of this element secures the public access. It is composed of:

  • Variable age which models the age and is of type int. The access is possible from the class of the same package.
  • Variable id which specifies a unique identifier and is of type String.
    • This element is visible from all the classes.
    • The element needs to be marked as final.
  • Constructor Animal(String, int) which initializes the unique identifier and age of an animal (assume the inputs will always be valid). The access of this type is public.
  • Method int getAge() which is getter. This element is visible from all the classes.
  • Method void setAge(int) which is setter. This element has the maximum visibility.
  • Method int score() which returns the sum of age and character length of unique identifier.
    • It is necessary to prohibit overriding.
    • This element is visible from all the classes.
  • Method String getId() which is getter. This element is visible from all the classes.

The class Cat is a cat. This element has the maximum visibility. It is a subtype of Animal. The class cannot be subclassed. It has:

  • Variable purrSound which specifies how the car purrs and is of type String. The element has the private visibility.
  • Constructor Cat(String, int, String) which initializes the unique identifier, age and cat barking. This element has the maximum visibility.
  • Method String getPurrSound() which is getter. Visibility of this element secures the public access.
  • Method void setPurrSound(String) which is setter. This element is visible from all the classes.

The class Dog is a dog. This element has the maximum visibility. The class cannot be subclassed. It extends the type Animal. It contains:

  • Variable barkSound which specifies how the dog barks and is of type String. The element is visible to clasess from the same paket as well as from the subclass.
  • Constructor Dog(String, int, String) which initializes the unique identifier, age and dog barking. Visibility of this element secures the public access.
  • Method String getBarkSound() which is getter. This element is visible from all the classes.
  • Method void setBarkSound(String) which is setter. The access of this type is public.

The class Calculator is an animal calculator. This element has the maximum visibility. It has:

  • Method double score(Animal) which calculates the animal score as follows. In case of a dog, returns the average between the age and character length of the barking. In case of a cat, returns the average between the character length of unique identifier and the character length of cat's purr sound. Important: in any other case (e.g. invalid input) returns 0 drugom slučaju vraća 0.
    • Visibility of this element secures the public access.
    • It is available in a static context of its own class.

Sample code:


		Dog dog1 = new Dog("Cathy", 4, "vau-vau-vau-vau");
		Dog dog2 = new Dog("Mary", 8, "VAU-VAUUU");
		
		Cat cat1 = new Cat("Kimi", 2, "mijau-mijau");
		Cat cat2 = new Cat("Niki", 3, "mijauu");
		Cat cat3 = new Cat("Mika", 1, "mijauuuuuuuuuuu");
		
		Animal[] animals = {dog1, dog2, cat1, cat2, cat3};
		
		for(Animal animal : animals) {
			System.out.printf("Animal %s has an animal score of %d and calculator score of %.2f %n", animal.getId(), animal.score(), Calculator.score(animal));
		}
