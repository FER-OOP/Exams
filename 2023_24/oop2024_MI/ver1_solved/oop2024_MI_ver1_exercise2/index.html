Opis zadatka

Napisati program koji modelira životinje (psa i mačku) te računa njihove ocjene. Ako nije drugačije specificirano, sve tipove je potrebno smjestiti u paket hr.fer.oop.

Razred Animal reprezentira životinju. Stvaranje objekata iz ove klase nije moguće. Vidljivost ovog elementa omogućuje javni pristup. U sebi sadržava:

  • Varijablu age koja specificira godine i tipa je int. Pristup je moguć iz klasa istog paketa.
  • Varijablu id koja modelira jedinstveni identifikator i tipa je String.
    • Element je vidljiv iz svih klasa.
    • Element mora biti označen kao finalan.
  • Konstruktor Animal(String, int) koji inicijalizira jedinstveni identifikator i godine životinje (pretpostaviti da će unosi uvijek biti ispravni). Pristup ovog tipa je javan.
  • Metodu int getAge() koja je getter. Element je vidljiv iz svih klasa.
  • Metodu void setAge(int) koja je setter. Element ima maksimalnu vidljivost.
  • Metodu int score() koja vraća zbroj godina i duljine znakova jedinstvenog identifikatora.
    • Potrebno je zabraniti nadjačavanje.
    • Element je vidljiv iz svih klasa.
  • Metodu String getId() koja je getter. Element je vidljiv iz svih klasa.

Razred Cat je mačka. Element ima maksimalnu vidljivost. Podtip je od Animal. Klasu nije moguće naslijediti. Ima:

  • Varijablu purrSound koja specificira kako se glasa mačka i tipa je String. Element ima privatnu vidljivost.
  • Konstruktor Cat(String, int, String) koji inicijalizira jedinstveni identifikator, godine i glasanje mačke. Element ima maksimalnu vidljivost.
  • Metodu String getPurrSound() koja je getter. Vidljivost ovog elementa omogućuje javni pristup.
  • Metodu void setPurrSound(String) koja je setter. Element je vidljiv iz svih klasa.

Klasa Dog je pas. Element ima maksimalnu vidljivost. Klasu nije moguće naslijediti. Nasljeđuje tip Animal. Sadrži:

  • Varijablu barkSound koja specificira kako laje pas i tipa je String. Element je vidljiv klasama iz istog paketa te klasama koje naslijeđuju taj element.
  • Konstruktor Dog(String, int, String) koji inicijalizira jedinstveni identifikator, godine i lajanje psa. Vidljivost ovog elementa omogućuje javni pristup.
  • Metodu String getBarkSound() koja je getter. Element je vidljiv iz svih klasa.
  • Metodu void setBarkSound(String) koja je setter. Pristup ovog tipa je javan.

Klasa Calculator je kalkulator životinja. Element ima maksimalnu vidljivost. Ima:

  • Metodu double score(Animal) koja računa ocjenu životinje na sljedeći način. Ako se radi o psu, vraća prosjek između godine psa i duljine znakova njegovog lajanja. Ako se radi o mački, vraća prosjek između duljine znakova jedinstvenog identifikatora i duljine znakova glasanja mačke. Važno: u bilo kojem drugom slučaju (npr. nepravilan unos) vraća 0.
    • Vidljivost ovog elementa omogućuje javni pristup.
    • Dostupna je u statičkom kontekstu svoje klase.

Primjer izvršavanja:


		Dog dog1 = new Dog("Cathy", 4, "vau-vau-vau-vau");
		Dog dog2 = new Dog("Mary", 8, "VAU-VAUUU");
		
		Cat cat1 = new Cat("Kimi", 2, "mijau-mijau");
		Cat cat2 = new Cat("Niki", 3, "mijauu");
		Cat cat3 = new Cat("Mika", 1, "mijauuuuuuuuuuu");
		
		Animal[] animals = {dog1, dog2, cat1, cat2, cat3};
		
		for(Animal animal : animals) {
			System.out.printf("Animal %s has an animal score of %d and calculator score of %.2f %n", animal.getId(), animal.score(), Calculator.score(animal));
		}
