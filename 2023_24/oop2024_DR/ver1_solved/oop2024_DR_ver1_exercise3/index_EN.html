The map players of type Map<String, Map<String, Integer>> consists of player names paired with their abilities (i.e. abilities are map of skills names paired with skills values). An example of data is given in the method DataLoader.loadPlayers and displayed at the end of this text.

You tasks are to finish two methods in class Task

  1. The method
Stream<PlayerInfo> skilfulPlayers(Map<String, Map<String, Integer>> players, Set<String> requiredSkills, int minSkillValue)

should return a stream of players with skill value greater or equal to minSkillValue in all required skills.

PlayerInfo is defined as

public record PlayerInfo(String name, int noOfSkillsDefined) {}

An example: if requiredSkills are [Positioning, Marking, Jumping, Strength] and minSkillValue is 12 then for the players in the example skilfulPlayers(players, skills, 12) should return a stream that contains (John Stone, 5) and (Mark Strong, 5) in any order.

  1. The method
Stream<PlayerInfo> bySkillAndName(Stream<PlayerInfo> players)

should return a stream in which players should be sorted first by skills count (in descending order), and then by name (if the skills count are the same).
