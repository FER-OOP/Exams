U zadatku su već zadane klase Pair i Main i njih ne smijete mijenjati. Vaš zadatak je napisati klasu LettersAndNumber (koja nasljeđuje klasu Pair) kako bi se program mogao uspješno prevesti i pokrenuti te dati rezultat kao što je navedeno u komentarima glavnog programa.

Poredak u skupu je po riječima (slovo u primjeru), a ako je riječ ista, onda se poredak određuje temeljem broja pridruženog riječi. S obzirom da su objekti tipa LettersAndNumber uvijek parovi (string, broj), za dohvat pridružene vrijednosti možete koristiti metodu doubleValue iz klase Number.

public class Pair<T, U> {
	private T first;
	private U second;
	
	public Pair(T first, U second) {
		this.first = first;
		this.second = second;
	}

	public T getFirst() {
		return first;
	}

	public void setFirst(T first) {
		this.first = first;
	}

	public U getSecond() {
		return second;
	}

	public void setSecond(U second) {
		this.second = second;
	}
}
