An excerpt from Croatian vascular flora is stored in file vrste.txt. Every line contains Latin name which consist of genus name, species name, authors' names and optionally subspecies name. Currently we are interested only in genus and species name which are the first two words in each line.

An example

Limonium bellidifolium (Gouan) Dumort.
Limonium cancellatum (Bernh. ex Bertol.) Kuntze
Limonium dictyophorum (Tausch) Degen
Limonium narbonense Mill.
Limonium pomoense Bogdanović et Brullo
Limonium subanfractum Trinajstić
Limonium vestitum (C. E. Salmon) C. E. Salmon
Limonium vestitum (C. E. Salmon) C. E. Salmon ssp. brusnicense Trinajstić
Limonium vestitum (C. E. Salmon) C. E. Salmon ssp. vestitum
Limonium virgatum (Willd.) Fourr.

With this we can see that genus Limonium has the following species bellidifolium, cancellatum, dictyophorum, narbonense, pomoense, subanfractum, vestitum, and virgatum .

Your task is to finish the method loadTaxa in the class DataLoader. The method should read the text file (its stream is the method's argument) and extracts genus and species names and returns a map that contains genera names as keys. Each key (i.e. genus) is paired with set of species names.

When iterating both map and set of species names it should be in alphabetical order.

Text is encoded using UTF-8, and you may assume that file is not empty, and all lines are correctly formatted.
