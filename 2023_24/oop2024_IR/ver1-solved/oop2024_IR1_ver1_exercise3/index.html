U zadatku je potrebno napraviti klasu TMWriter koja direktno nasljeđuje apstraktnu klasu Writer. Klasa TMWriter se koristi kao dekorator neke druge klase koja nasljeđuje klasu Writer, a pretvara hrvatske dijaktritike u odgovarajuće digrame kako je to u 19. stoljeću predlagao hrvatski jezikoslovac Tomislav Maretić (TM). Da je Maretićev prijedlog tada bio prihvaćen, tekstovi na hrvatskom jeziku bi se danas lako mogli unositi na stranim tipkovnicama (engleska, njemačka, francuska itd.).

Popis dijakritika i digrama kojima ih je potrebno zamijeniti:

  • "č" -> "cx"
  • "Č" -> "Cx"
  • "ć" -> "cy"
  • "Ć" -> "Cy"
  • "dž" -> "dx"
  • "Dž" -> "Dx"
  • "đ" -> "dy"
  • "Đ" -> "Dy"
  • "nj" -> "ny"
  • "Nj" -> "Ny"
  • "lj" -> "ly"
  • "Lj" -> "Ly"
  • "š" -> "sx"
  • "Š" -> "Sx"
  • "ž" -> "zx"
  • "Ž" -> "Zx"

NAPOMENA: Klasa TMWriter ne smije biti finalna tj. mora se moći naslijediti radi ispravnog bodovanja rješenja.

Primjer korištenja ove klase prikazan je u sljedećem isječku dane metode main u klasi Main:

//print original
System.out.println("Original:");
System.out.println(LOWER_CASE_EXAMPLE);
System.out.println(UPPER_CASE_EXAMPLE);

//writer
StringWriter sWriter = new StringWriter(); //to simplify, we will read the text from a string rather than from a file
TMWriter tmWriter = new TMWriter(sWriter); //TMWriter decorates the StringWriter
BufferedWriter bWriter = new BufferedWriter(tmWriter);
bWriter.write(LOWER_CASE_EXAMPLE);
bWriter.newLine();
bWriter.write(UPPER_CASE_EXAMPLE);
bWriter.flush();

//print tm style
System.out.println("TM Style:");
System.out.println(sWriter.toString());

Ispravan rezultat izvođenja dane metode main u klasi Main:

Original:
Franc Hlapić smješta ključ od gvožđa uz džbunje.
franc hlapiĆ smjeŠta kLjuČ od gvoŽĐa uz DžbuNje.
TM Style:
Franc Hlapicy smjesxta klyucx od gvozxdya uz dxbunye.
franc hlapiCy smjeSxta kLyuCx od gvoZxDya uz DxbuNye.

NAPOMENA: Sve klase i sučelja imaju vidljivost postavljenu na public i nalaze se u paketu hr.fer.oop.
