U zadatku je potrebno nadopuniti programski kod klase Item te napisati klasu SpeciallLinkedList.

Klasa Item s atributima name, value i category modelira fizički predmet pri čemu niti jedan od ova 3 atributa nije jedinstven. Ova klasa implementira sučelje Comparable<Item>. U ovoj klasi je potrebno implementirati metode hashCode, equals i compareTo. Prirodni poredak instanci ove klase je abecedno po imenu (tj. vrijednosti atributa name). Ako dva fizička predmeta imaju isto ime, nebitno je koji će od njih biti ispred drugog u prirodnom poretku.

Parametrizirana klasa SpecialLinkedList<E extends Comparable<E>> nasljeđuje klasu java.util.LinkedList<E> te implementira dano sučelje SpecialCollection<E>. Za razliku od sučelja java.util.List, sučeljem SpecialCollection se modeliraju liste koje ne smiju sadržavati null elemente niti duplikate. Prilikom dodavanja null elementa, metode ove klase trebaju bacati iznimku tipa java.lang.NullPointerException, a prilikom dodavanja elementa koji je već u kolekciji iznimku tipa java.lang.IllegalArgumentException. Zbog toga je u klasi SpecialLinkedList potrebno nadjačati samo one metode iz njene nadklase LinkedList u kojima treba izazvati odgovarajuću iznimku zbog dodavanja null elementa ili duplikata. Osim toga, u klasi SpecialLinkedList je potrebno implementirati i metodu getSorted iz sučelja SpecialCollection koja vraća novi objekt tipa SpecialCollection koji sadrži sve elemente iz originalne kolekcije, ali sortirane prirodnim poretkom.

NAPOMENA: Zanemarite činjenicu da je u listu moguće dodati elemente pozivanjem metoda add(E e) i set(E e) nad objektom tipa ListIterator. Ne trebate izazivati iznimke prilikom dodavanja null elementa i duplikata na ovaj način.

Primjer korištenja ove klase prikazan je u sljedećem isječku dane metode main u klasi Main:

SpecialCollection<Item> itemCollection = new SpecialLinkedList<>();

Item item1 = new Item("Sword", 300, "Weapon");
Item item2 = new Item("Shield", 150, "Armor");
Item item3 = new Item("Potion", 50, "Consumable");
Item item4 = new Item("Helmet", 100, "Armor");
Item item5 = new Item("Sword", 300, "Weapon"); // Duplicate
Item item6 = null; // Null-value

itemCollection.add(item1);
itemCollection.add(item2);
itemCollection.add(item3);
itemCollection.add(item4);
//itemCollection.add(item5); // Should throw an IllegalArgumentException
//itemCollection.add(item6); // Should throw a NullPointerException

System.out.println("\nAll items:");
System.out.println(itemCollection);        

System.out.println("\nRemoving item 'Shield':");
itemCollection.remove(item2);
System.out.println(itemCollection);
        
System.out.println("\nAll items sorted:");
System.out.println(itemCollection.getSorted());

Ispravan rezultat izvođenja dane metode main u klasi Main:

    Removing item 'Shield':
    [Item{name=Sword, value=300, category=Weapon}, Item{name=Potion, value=50, category=Consumable}, Item{name=Helmet, value=100, category=Armor}]

    All items sorted:
    [Item{name=Helmet, value=100, category=Armor}, Item{name=Potion, value=50, category=Consumable}, Item{name=Sword, value=300, category=Weapon}]
