In this assignment, it is necessary to create a class TMWriter that directly extends the abstract class Writer. The class TMWriter is used as a decorator for another class that extends the Writer class, converting Croatian diacritics into corresponding digraphs as proposed by Croatian linguist Tomislav Maretić (TM) in the 19th century. If Maretić's proposal had been accepted at the time, texts in the Croatian language could easily be entered on foreign keyboards (English, German, French, etc.).

The list of diacritics and the digraphs they should be replaced with:

  • "č" -> "cx"
  • "Č" -> "Cx"
  • "ć" -> "cy"
  • "Ć" -> "Cy"
  • "dž" -> "dx"
  • "Dž" -> "Dx"
  • "đ" -> "dy"
  • "Đ" -> "Dy"
  • "nj" -> "ny"
  • "Nj" -> "Ny"
  • "lj" -> "ly"
  • "Lj" -> "Ly"
  • "š" -> "sx"
  • "Š" -> "Sx"
  • "ž" -> "zx"
  • "Ž" -> "Zx"

NOTE: The class TMWriter must not be final, i.e., it must be inheritable for correct scoring of the solution.

An example of using this class is shown in the following snippet of the given main method in the Main class:

//print original
System.out.println("Original:");
System.out.println(LOWER_CASE_EXAMPLE);
System.out.println(UPPER_CASE_EXAMPLE);

//writer
StringWriter sWriter = new StringWriter(); //to simplify, we will read the text from a string rather than from a file
TMWriter tmWriter = new TMWriter(sWriter); //TMWriter decorates the StringWriter
BufferedWriter bWriter = new BufferedWriter(tmWriter);
bWriter.write(LOWER_CASE_EXAMPLE);
bWriter.newLine();
bWriter.write(UPPER_CASE_EXAMPLE);
bWriter.flush();

//print tm style
System.out.println("TM Style:");
System.out.println(sWriter.toString());

The correct output of the given main method in the Main class should be:

Original:
Franc Hlapić smješta ključ od gvožđa uz džbunje.
franc hlapiĆ smjeŠta kLjuČ od gvoŽĐa uz DžbuNje.
TM Style:
Franc Hlapicy smjesxta klyucx od gvozxdya uz dxbunye.
franc hlapiCy smjeSxta kLyuCx od gvoZxDya uz DxbuNye.
