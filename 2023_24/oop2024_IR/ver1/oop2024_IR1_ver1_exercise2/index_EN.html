In this assignment, it is necessary to supplement the program code of the class Item and write the class SpecialLinkedList.

The class Item with attributes name, value, and category models a physical item where none of these 3 attributes are unique. This class implements the interface Comparable<Item>. In this class, it is necessary to implement the methods hashCode, equals, and compareTo. The natural ordering of instances of this class is alphabetical by name (i.e., the value of the name attribute). If two physical items have the same name, it does not matter which one comes before the other in the natural ordering.

The parameterized class SpecialLinkedList<E extends Comparable<E>> inherits the class java.util.LinkedList<E> and implements the given interface SpecialCollection<E>. Unlike the interface java.util.List, the interface SpecialCollection models lists that must not contain null elements or duplicates. When adding a null element, the methods of this class should throw an exception of type java.lang.NullPointerException, and when adding an element that is already in the collection, an exception of type java.lang.IllegalArgumentException. Therefore, in the class SpecialLinkedList, it is necessary to override only those methods from its superclass LinkedList where it is necessary to trigger the appropriate exception due to adding a null element or a duplicate. Additionally, in the class SpecialLinkedList, it is necessary to implement the getSorted method from the SpecialCollection interface, which returns a new object of type SpecialCollection that contains all elements from the original collection, but sorted in natural order.

NOTE: Ignore the fact that it is possible to add elements to the list by calling the methods add(E e) and set(E e) on an object of type ListIterator. You do not need to trigger exceptions when adding a null element and duplicates in this way.

An example of using this class is shown in the following snippet of the given main method in the Main class:

SpecialCollection<Item> itemCollection = new SpecialLinkedList<>();

Item item1 = new Item("Sword", 300, "Weapon");
Item item2 = new Item("Shield", 150, "Armor");
Item item3 = new Item("Potion", 50, "Consumable");
Item item4 = new Item("Helmet", 100, "Armor");
Item item5 = new Item("Sword", 300, "Weapon"); // Duplicate
Item item6 = null; // Null-value

itemCollection.add(item1);
itemCollection.add(item2);
itemCollection.add(item3);
itemCollection.add(item4);
//itemCollection.add(item5); // Should throw an IllegalArgumentException
//itemCollection.add(item6); // Should throw a NullPointerException

System.out.println("\nAll items:");
System.out.println(itemCollection);        

System.out.println("\nRemoving item 'Shield':");
itemCollection.remove(item2);
System.out.println(itemCollection);
        
System.out.println("\nAll items sorted:");
System.out.println(itemCollection.getSorted());
