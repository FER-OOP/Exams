Note: If you implemented the University class in the 2nd task, you can use it here as well. Just write the code that is additionally required.

You need to write a University class that describes an university. Each university has the following attributes:

name - the name of the university

country - the name of the country where the university is located

founded - year of establishment (integer number)

therank - the current rank of the university on the Times Higher Education World University Rankings list (integer number)

arwurank - the current ranking of the university according to the Academic Ranking of World Universities, the so-called "Shanghai list" (integer)

All attributes are private and for each one it is necessary to create a getter method. The getter names should be correct, eg the getter for the name attribute should be getName().

It is necessary to define a constructor that will create a new object of University type based on all the attributes, as listed above, and override the toString() method as follows:

public String toString() {
	return String.format("%s (%s, %d) THE: %d ARWU: %d", name, country, founded, therank, arwurank);
}

Implement the methods required for the University class to be used correctly in Java collections (ArrayList, LinkedList, HashSet, TreeSet).

Bare in mind that two universities are equal if they have the same name, the name of the country in which university is located, and the year of foundation.

Define a natural comparator that will order universities first by name (from A to Z), then by country name (from A to Z) and finally by year of establishment (newer universities first, then older ones).

Define public, static and final variables that will represent additional comparators for individual attributes:

BY_NAME - comparator by name, from smallest to largest (A to Z)

BY_COUNTRY – comparator by country name (A to Z)

BY_THE – comparator by the Times Higher Education World University Rankings list, from smallest to largest

BY_ARWU – comparator by Academic Ranking of World Universities list, from smallest to largest

Note: The class `University' should be public and should be placed in the package hr.fer.oop.jir2024.zad4.
