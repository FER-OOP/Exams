The map players of type Map<String, Map<String, Integer>> players consists of player names paired with their abilities (i.e. abilities are map of skills names paired with skills values). An example of data is given in the the method DataLoader.loadPlayers and displayed at the end of this text.

You tasks are to finish two methods in class Task

  1. The method
int countSkills(Map<String, Integer> playerSkills, Set<String> skills, int minSkillValue)

will seek for skills in playerSkills, and return how many of them has value greater of equal to minSkillValue

An example: if playersSkills is {Positioning=7, Speed=20, Shooting=20, Marking=20, Jumping=20, Tackling=8, Strength=4} and skills are [Positioning, Marking, Finishing, Tackling, Strength] then countSkills(playersSkills, skills, 8) should return 2 (because only Tackling and Marking have values greater or equal to 8)

  1. The method
List<String> bestPlayers(Map<String, Map<String, Integer>> players, Set<String> skills, int minSkillValue, double minPercentage)

will use method countSkills to return players names which has at least minPercentage of skills with value greater or equal to minSkillValue.

An example: if skills are [Positioning, Marking, Jumping, Tackling, Strength] then for the players in the example bestPlayers(playersSkills, skills, 15, 0.8) should return John Stone and Mark Strong (because they have at least 4 of these 5 skills with values greater or equal to 15)
